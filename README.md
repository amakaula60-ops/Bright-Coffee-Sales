# Bright-Coffee-Sales

1. Objective
The primary objective of this project was to derive actionable business insights from the Bright Coffee Shop Sales dataset to support the new CEO’s strategic decision-making. The analysis aimed to uncover:
•	Products generating the highest and lowest revenue
•	Peak sales hours and performance by time intervals
•	Overall sales trends across products and time
•	Recommendations to enhance sales and operational performance
The end goal was to translate raw transactional data into clear, visual, and data-driven insights that could guide the CEO in growing revenue and optimizing product offerings.

2. Tools Used	
Miro	
Purpose - Planning and architecture design for data flow.
Output - Data Architecture Diagram & Insight Planning.

Snowflake	
Purpose - Data storage, cleaning, and transformation using SQL queries.	
Output - Processed dataset ready for visualization.

Microsoft Excel	
Purpose - Data visualization using Pivot Tables and Graphs for analysis and storytelling.
Output - Dashboards, Charts & Pivot Summaries.

Microsoft PowerPoint	
Purpose - Final presentation to communicate findings, visuals, and recommendations.
Output - Executive Presentation Slides

3. Project Plan and Workflow
Phase 1: Planning & Architecture (Miro)
Phase 2: Data Processing (Snowflake)
Phase 3: Data Analysis & Visualization (Excel)
Phase 4: Executive Presentation (PowerPoint)

4. Outcome Summary
•	The top-selling products generated the majority of revenue, particularly during morning and midday time slots.
•	Sales peaked between 08:00–11:00, aligning with customer coffee-buying patterns.
•	Underperforming products were identified, providing opportunities for targeted promotions.
•	Recommendations included:
    o	Launching marketing campaigns during off-peak hours.
    o	Increasing stock of best-selling products.
    o	Introducing loyalty programs for frequent morning customers.

5. Conclusion
This project successfully demonstrated a full analytics workflow — from data architecture design to insight presentation — using real transactional data.
The outputs provided the CEO with an evidence-based understanding of product and sales performance, offering clear strategic recommendations for business growth.
